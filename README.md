# holosynth: 开发文档

本文档旨在阐述 HoloSynth 音频合成引擎底层的物理声学模型、数字信号处理（DSP）算法以及系统运行架构。本引擎属于确定性（Deterministic）的离散时间音频系统，通过多维张量运算实现波形合成、空间声像定位与线性时不变（LTI）空间混响。

---

## 一、 信号发生与合成模型 (Signal Generation & Synthesis)

引擎的振荡器核心基于相位调制（Phase Modulation, 广义FM合成）与离散时间滤波网络构建。

### 1.1 十二平均律与基频映射 (Equal Temperament Mapping)
系统默认采用 $A_4 = 440\text{Hz}$ 为标准参考频率。对于给定调式内的任意音符索引，其离散物理频率 $f$ 的映射公式为：
$$f(k, o) = f_{ref} \cdot 2^{\frac{-9}{12}} \cdot 2^{\frac{R}{12}} \cdot 2^{o + \frac{k_{shift}}{12}}$$
其中：
* $R$ 为根音偏移量（Root Offset，如 $C=0, C^\sharp=1$）。
* $o$ 为八度标量（Octave）。
* $k_{shift}$ 为对应音符在当前音阶矩阵中的半音偏移度数。

### 1.2 时变相位调制合成 (Time-Variant Phase Modulation)
为提升运算效率并生成复杂的非谐波泛音列，系统采用相位调制方程生成单声部波形。对于离散时间索引 $n$，生成的信号 $y[n]$ 满足：
$$y[n] = A \cdot \sin\left(\phi_c[n] + I[n] \cdot \sin(\phi_m[n])\right) + v[n]$$
其中：
* $\phi_c[n] = \sum_{i=0}^{n} 2\pi \frac{f_c}{f_s}$ 为载波（Carrier）累加相位，$f_s$ 为采样率。
* $\phi_m[n] = \sum_{i=0}^{n} 2\pi \frac{f_c \cdot R_{fm}}{f_s}$ 为调制波（Modulator）累加相位，$R_{fm}$ 为频率比。
* $I[n]$ 为时变的调制指数（Modulation Index）。在引擎中，$I[n]$ 为在声音持续期内线性插值的数组，用于实现动态的音色频谱演变。
* $v[n] \sim \mathcal{N}(0, \sigma^2)$ 为叠加的高斯白噪声，用于模拟气流或打击乐瞬态。

### 1.3 双二阶共振低通滤波器 (Biquadratic Resonant LPF)
合成后的宽带信号将通过一个二阶 IIR（无限脉冲响应）滤波器进行减法塑形。其在 Z 域的传递函数为：
$$H(z) = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2}}{a_0 + a_1 z^{-1} + a_2 z^{-2}}$$
系统采用 Audio EQ Cookbook 标准设计，给定截止频率 $f_c$ 与共振品质因数 $Q$，归一化角频率 $\omega_0 = 2\pi \frac{f_c}{f_s}$，且 $\alpha = \frac{\sin(\omega_0)}{2Q}$。滤波器系数计算如下：
$$b_0 = \frac{1 - \cos(\omega_0)}{2}, \quad b_1 = 1 - \cos(\omega_0), \quad b_2 = \frac{1 - \cos(\omega_0)}{2}$$
$$a_0 = 1 + \alpha, \quad a_1 = -2\cos(\omega_0), \quad a_2 = 1 - \alpha$$
该滤波器以 $-12\text{dB/octave}$ 的斜率衰减高于 $f_c$ 的频率分量，并在 $f_c$ 处产生由 $Q$ 决定的极点增益。

## 二、 空间音频与声像定位算法 (Spatial Audio & Psychoacoustics)

系统通过模拟双耳声级差（ILD）与双耳时间差（ITD），构建具有物理真实感的 3D 声场与多普勒效应。

### 2.1 距离衰减 (Inverse Distance Attenuation)
声波能量随距离增加而衰减。系统引入无量纲距离参数 $d[n]$，距离增益函数定义为：
$$G_{dist}[n] = \frac{1}{1 + d[n]}$$

### 2.2 恒定功率声像分配 (Constant-Power Panning)
为防止声源在声相中心（Pan Center）移动时发生功率塌陷，引擎依据等距能量守恒定律分配左右声道增益。设方位角为 $\theta[n]$，在 X 轴的线性投影比例为 $p[n] = \frac{\sin(\theta[n]) + 1}{2}$，则声道增益为：
$$G_L[n] = \cos\left(p[n] \cdot \frac{\pi}{2}\right)$$
$$G_R[n] = \sin\left(p[n] \cdot \frac{\pi}{2}\right)$$
易证 $G_L^2[n] + G_R^2[n] \equiv 1$，确保总输出声功率与空间方位无关。

### 2.3 分数延迟与多普勒效应 (Fractional Delay & Doppler Shift)
系统通过可变延迟线（Variable Delay Line）模拟声音到达双耳的时间差（ITD）。最大头颅延迟设定为 $\tau_{max}$（约 $0.6\text{ms}$，对应 $44100\text{Hz}$ 下 $26$ 个采样点）。
左右耳的延迟采样数为：
$$\tau_L[n] = \tau_{max} \cdot \max(0, \sin(\theta[n]))$$
$$\tau_R[n] = \tau_{max} \cdot \max(0, -\sin(\theta[n]))$$
为消除离散采样导致的量化噪声（Zipper Noise），系统采用一阶线性插值（Linear Interpolation）计算非整数延迟索引下的幅值：
$$y_{L,R}[n] = x[n - \tau_{L,R}[n]] \cdot G_{L,R}[n] \cdot G_{dist}[n]$$
**多普勒物理推导**：当声源向观察者靠近时，距离/角度的改变导致 $\tau[n]$ 随时间递减，即时间导数 $\frac{d\tau}{dt} < 0$。读取指针遍历波形的速度将大于波形的生成速度，产生物理挤压，等效于频率蓝移（增高）：
$$f_{obs} = f_{src} \left( 1 - \frac{d\tau}{dt} \right)$$

## 三、 线性时不变系统与卷积混响 (LTI Systems & Convolution Reverb)

对于空间环境的反射声学建模，系统通过生成合成房间脉冲响应（IR），并在频域内进行快速卷积运算实现。

### 3.1 脉冲响应生成 (Impulse Response Generation)
房间脉冲 $h[n]$ 由高频阻尼白噪声与指数衰减包络构成。设混响尾音时间常数为 $L = f_s \cdot T_{room}$：
$$h_0[n] = w[n] \cdot e^{-5.0 \frac{n}{L}}, \quad \text{其中 } w[n] \sim \mathcal{N}(0, 1)$$
随后，$h_0[n]$ 通过一阶低通滤波器以模拟空气对高频声波的吸收（Damping）。设衰减系数为 $\gamma$：
$$H_{damp}(z) = \frac{1 - \gamma}{1 - \gamma z^{-1}}$$
$$h[n] = h_0[n] * \mathcal{Z}^{-1}\{H_{damp}(z)\}$$

### 3.2 快速傅里叶卷积 (Fast Fourier Convolution)
依据卷积定理，时域中的离散卷积等价于频域中的逐元素乘积。干声信号 $x[n]$ 与 $h[n]$ 的混响场 $y_{wet}[n]$ 的计算避免了 $O(N \cdot M)$ 的时域开销，优化为 $O(N \log N)$ 的频域操作：
$$y_{wet}[n] = \mathcal{F}^{-1}\left\{ \mathcal{F}\{x[n]\} \cdot \mathcal{F}\{h[n]\} \right\}$$
最终输出由干湿比（Wet Level）常量 $W$ 进行线性混合：
$$y_{out}[n] = (1 - W)x[n] + W y_{wet}[n]$$

## 四、 系统生命周期与并行运算逻辑 (Execution Flow)

当调用 `Synthesizer.mix()` 时，引擎依据以下流水线进行批处理运算：

1. **矩阵向量化 (Vectorization)**: 解析每一行的声学参数，依据节拍时值 $T$，利用等差数列（`np.linspace`）将标量参数（频率、位置、FM指数）升维至长度为 $N = f_s \cdot T$ 的一维张量。
2. **相位接续 (Phase Accumulation & Memory)**: 若前置音符未触发释放（Release）阶段，系统提取前置音符的末端相位 $\phi[N-1]$ 作为当前音符的初始相位 $\phi_0$，确保连续波形的 $C^0$ 连续性，从源头抑制瞬态爆音。
3. **幅值调制 (Amplitude Modulation)**: 使用用户定义的控制点，通过线性插值生成全局 ADSR 归一化包络张量 $E[n] \in [0,1]$，与空间化后的双声道波形逐元素相乘。
4. **波形干涉 (Waveform Interference)**: 多条相互独立的音轨生成定长张量后，在时域上进行标量加法操作（$\sum_{i} y_i[n]$），模拟声波在物理空间中的线性叠加律。
5. **峰值归一化 (Peak Normalization)**: 对混合后的总线波形执行遍历，提取绝对极大值 $A_{max} = \max|y_{mix}[n]|$。执行前馈式硬压限（Hard Limiter）：
   $$y_{final}[n] = \frac{y_{mix}[n]}{A_{max}} \cdot 32767$$
   最终强制类型转换为 `int16` 脉冲编码调制（PCM）数据流。

## 五、 参数物理映射字典 (Parameter Dictionary)

| 参数 | 物理/数学定义 | 理论值域 | 信号学影响 |
| :--- | :--- | :--- | :--- |
| **`angle`** | 极坐标系下的相对方位角 $\theta$。 | $[0, 360]$ | 改变 ITD 相位差与 ILD 振幅映射。支持超界（如 $1080$）用于连续旋量计算。 |
| **`distance`** | 与听觉坐标系原点的相对距离标量 $d$。| $[0, +\infty)$ | $d=0$ 时为无穷近理想点声源。决定距离衰减系数的倒数。 |
| **`fm_ratio`** | 频率调制倍频程比 $R_{fm} = f_m / f_c$。| $\mathbb{R}^+$ | 控制贝塞尔函数产生的边带（Sidebands）间距。整数呈谐波，小数呈金属失谐。 |

| **`q_factor`** | 滤波器谐振品质因数 $Q$。| $[0.1, 20.0]$ | 决定在 $f_c$ 处的能量聚集程度。高值引发强烈的自激振荡边缘效应。 |
| **`noise`** | 高斯白噪声的方差参数 $\sigma$。| $[0.0, +\infty)$ | 产生覆盖所有频段的宽带随机能量，用于拟真气流噪声或物理瞬态。 |
| **`room_size`** | 混响脉冲指数衰减的对数时间常数。| $[0.1, 5.0]$ | 在时域上延长脉冲响应长度，宏观上决定听觉空间在物理上的"容积"大小。 |
| **`damping`** | 一阶低通混响反馈系数 $\gamma$。| $[0.0, 1.0)$ | 改变脉冲的高频滚降（Roll-off）斜率，模拟高比热容物理材料的声学吸收特性。 |
## 六、 参数声学感知与编程实战 (Acoustic Perception & Programming Guide)

在实际的数字音频工作站（DAW）操作与本引擎的矩阵编程中，数学参数的改变将直接引发人类听觉心理学的剧烈变化。本节将详细说明各个核心参数的“听感效应”以及对应的 Python 矩阵编写语法。

### 6.1 空间感知参数 (Spatial Perception)

在矩阵中，空间参数位于第 4 个维度：`[角度, 距离]`。引擎支持传入元组 `(start, end)` 来实现基于时间的平滑积分计算。

* **`angle` (方位角)**
  * **声学变化**：控制声音在水平声相（Panning）中的位置。`0` 为眉心正前方，`90` 为右耳，`270` 为左耳。
  * **高阶听感**：当 `angle` 设置为元组时，由于底层 ITD（双耳时间差）插值的存在，你不仅会听到声音在左右移动，还会产生明显的“多普勒频移（声音靠近时变尖锐，远离时变低沉）”。
  * **编程示例**：
    ```python
    # 声音在 4 拍内，从左后方 (225度) 丝滑旋转到正右方 (90度)
    # 语法：[音符, 八度, 节拍, [角度轨迹, 距离], 包络, 音色]
    [0, 0, 4.0, [(225, 90), 2.0], Presets.Envelopes.PAD, Presets.Instruments.SINE_PURE]
    ```

* **`distance` (声源距离)**
  * **声学变化**：数字越大，声音的干声音量越小。
  * **高阶听感**：当配合卷积混响（Reverb）使用时，改变 `distance` 不仅是把声音“关小”，而是会让声音被混响的尾音完全吞噬，产生极强的“深渊感”和“宏大感”。
  * **编程示例**：
    ```python
    # 距离从极远处 (20) 瞬间逼近到贴脸 (0)，产生强烈的 Jump Scare (突发惊吓) 压迫感
    [0, 0, 1.0, [0, (20, 0)], Presets.Envelopes.SWELL, Presets.Instruments.DARK_BRASS]
    ```

### 6.2 音色调制参数 (Timbre & FM Synthesis)


在矩阵的第 6 个维度 `synth_params` 中，前三个参数 `[amp, fm_ratio, fm_index]` 构成了声音的骨架。

* **`fm_ratio` (频率比)**
  * **声学变化**：决定声音是“乐器”还是“敲击金属”。
    * 设为 `1.0` 或 `2.0`：产生规则的谐波，听起来像拉弦乐器、管乐器或经典的模拟合成器。
    * 设为非整数（如 `2.77` 或 `3.5`）：产生非谐波泛音列，听起来像敲击玻璃、大钟、或者木鱼。
* **`fm_index` (调制深度)**
  * **声学变化**：决定声音的“亮暗”与“毛刺感”。指数越高，高频泛音越丰富，声音越刺耳。
  * **高阶听感**：使用 `(start, end)` 元组可以创造“随时间闭合”的音色。这模仿了用力砸下钢琴按键瞬间高频极其明亮，随后高频迅速消散的物理现象。
  * **编程示例**：
    ```python
    # 制作极其逼真的 80 年代 DX7 电子钟声
    # ratio=3.5 (金属感), index=(6.0, 0.0) (从极其刺耳瞬间衰减为纯净正弦波)
    # 语法: [amp, fm_ratio, fm_index, cutoff_hz, q_factor, noise]
    synth_bell = [0.8, 3.5, (6.0, 0.0), 20000, 0.7, 0.0]
    [0, 1, 1.0, [0, 1], Presets.Envelopes.PLUCK, synth_bell]
    ```

### 6.3 频段雕刻与瞬态 (Filter & Noise)


这是 EDM (电子舞曲) 与现代合成器中最具辨识度的参数：`[..., cutoff_hz, q_factor, noise]`。

* **`cutoff_hz` (截止频率)**
  * **声学变化**：像一扇门一样控制高频的放行。`20000` 时门全开，声音最亮；调到 `800` 时，声音像是被捂在了被子里（Muffled）。
* **`q_factor` (共振 Q 值)**
  * **声学变化**：在截止频率处产生一个声音的“尖刺”。Q 值大于 `3.0` 时，声音会带有一种强烈的“水滴声”、“吞咽声”或电子科幻感（常被称为 Acid / 哇音）。
  * **编程示例**：
    ```python
    # 制作经典的 TB-303 酸性贝斯 (Acid Bass)
    # 使用 5.0 的超高 fm_index 制造极度刺耳的波形，然后用 800Hz 截止频率捂住，并附加 5.0 的超高 Q 值激发出“哇哇”声
    synth_acid = [0.6, 1.0, 5.0, 800, 5.0, 0.0]
    [0, -2, 0.25, [0, 0], Presets.Envelopes.STACC, synth_acid]
    ```

* **`noise` (高斯白噪声)**
  * **声学变化**：给纯净的声波撒上“灰尘”。
  * **微量 (`0.02 - 0.1`)**：模拟老式黑胶唱片的底噪，或者磁带机的电流声（Lo-Fi 风格）。
  * **极大量 (`1.0 - 5.0`)**：彻底掩盖原本的音高，产生剧烈的风暴声或爆炸声。
  * **编程示例**：
    ```python
    # 制作科幻感十足的“气流喷射军鼓”
    # 关闭 FM 调制，将白噪声拉到 3.0，配合极短的 PLUCK 包络产生瞬间的爆破音
    synth_snare = [0.8, 1.0, 0.0, 20000, 0.7, 3.0]
    [0, 0, 0.5, [0, 0], Presets.Envelopes.PLUCK, synth_snare]
    ```

### 6.4 时域包络控制 (Envelope ADSR Logic)


在矩阵的第 5 个维度中，包络线通过形如 `[[时间比例, 音量比例], ...]` 的散点坐标构成。

* **断奏感 (Staccato / Pluck)**
  * **物理特性**：极短的起音（Attack），极快的衰减（Decay），能量不持续。
  * **写法**：`[[0.0, 0], [0.05, 1], [0.2, 0]]`。声音在 5% 的时间点达到最大音量，在 20% 的时间点彻底消失，剩余 80% 的时间保持死寂。
* **延音/连音感 (Legato / Pad)**
  * **物理特性**：缓慢像雾一样升起，并且在音符结束时不归零，以便与下一行矩阵的波形相位完美缝合。
  * **写法**：`[[0.0, 0], [0.5, 1], [1.0, 1]]`。音量到最后保持在 1.0（不切断）。
  * **⚠️ 编程警告**：如果你使用了不归零的包络，**下一行绝对不能是休止符 `[-1]`**。必须通过一个收尾包络（如 `[[0.0, 1], [1.0, 0]]`）进行收束，否则会导致波形突变引发喇叭劈啪爆音（Clipping）。

### 6.5 全局混响参数 (Global Reverb Settings)

混响不在矩阵中控制，它通过 `synth.apply_reverb(audio, room_size, damping, wet_level)` 挂载到渲染完毕的全局母带上。

* **`room_size` (房间尺寸)**
  * **`0.5` ~ `1.0`**：小型录音室或浴室。增加声音的厚度和真实感，但不拖泥带水，适合极速的贝斯和军鼓。
  * **`2.0` ~ `3.0`**：大型音乐厅或大教堂。赋予声音史诗感。
  * **`4.0+`**：无边无际的星际深渊或大峡谷。适合环境音乐（Ambient / Pad）。
* **`damping` (高频阻尼吸收)**
  * **`0.1`**：房间墙壁极其光滑（如玻璃、大理石），回声极其清脆、刺耳。
  * **`0.8`**：房间挂满了吸音海绵和地毯。高频被迅速吃掉，回声听起来非常深邃、温暖且“发闷”。
* **`wet_level` (混响声音占比)**
  * **`0.01`**：几乎没有混响，声音很干。
  * **`0.8`**：混响占比高，音频空间感比较强烈，有一种回声空灵通透的感觉
* **编程示例**：
  ```python
  # 渲染出干声轨道
  dry_audio = synth.mix([track_1, track_2])
  
  # 挂载史诗级的黑暗深渊混响：
  # 房间巨大(3.5秒)，空气沉闷(阻尼0.8吸收了所有高频)，干湿比0.4(保留一部分原本的清晰度)
  epic_master = synth.apply_reverb(dry_audio, room_size=3.5, damping=0.8, wet_level=0.4)

## 七、 案例：构建全息交响曲 (Ultimate Masterpiece: Building from Scratch)



在掌握了所有的理论后，我们将不依赖任何内置的 `Presets`，纯手工用底层数组构建一首极具科幻感的 3D 空间短曲《奇点 (Singularity)》。

通过剥离预设，你可以直观地看到：**音乐、音色、空间、时间，最终在计算机底层不过是一堆精确排列的浮点数数组。**

### 完整演示代码 (`singularity_demo.py`)

```python
"""
PyHoloSynth 终极实战演示: 《奇点 (Singularity)》
本代码完全不使用任何 Presets 预设，展示引擎最硬核的底层参数化控制力。
"""
from holosynth import Synthesizer

print("🚀 正在初始化《奇点》全息声场计算核心...")

# ========================================================
# 1. 核心引擎初始化 (自定义外星音阶)
# ========================================================
# 抛弃标准的自然大/小调，我们手写一个极具异域悬疑感的“双和声小调”
# 标准 Presets.Scales.MINOR 是 [0, 2, 3, 5, 7, 8, 10]
CUSTOM_SCALE = [0, 1, 4, 5, 7, 8, 11] 
synth = Synthesizer(bpm=100, scale_intervals=CUSTOM_SCALE)

# ========================================================
# 2. 纯手工定义声学参数 (解析 Presets 的本质)
# ========================================================
# 格式: [振幅, FM频率比, FM深度, 截止频率, Q值, 噪声]
# 手写音色 A: 虚空低音 (取代 Presets.Instruments.WARM_PAD)
# 原理: 极低的截止频率 (600Hz) 产生沉闷感，轻微的 FM 比率 (1.01) 产生失真偏频。
CUSTOM_SUB_BASS = [0.6, 1.01, 1.0, 600, 1.5, 0.0]

# 手写音色 B: 神经突触电报音 (取代 Presets.Instruments.CHIPTUNE)
# 原理: 极高的 FM 深度 (动态从 10.0 衰减到 0.0)，高共振 Q值 (4.0)，产生尖锐的滴答声。
CUSTOM_GLITCH   = [0.4, 3.5, (10.0, 0.0), 5000, 4.0, 0.0]

# 手写音色 C: 太阳风暴 (取代 Presets.Instruments.NOISE_WIND)
# 原理: 完全关闭主振幅，将噪声拉满 (2.0)，利用低通滤波器 (3000Hz) 压制高频刺耳声。
CUSTOM_WIND     = [0.0, 1.0, 0.0, 3000, 2.0, 2.0]

# 手写包络线 (取代 Presets.Envelopes)
# 格式: [[时间比例, 音量比例], ...]
ENV_SWELL = [[0.0, 0.0], [0.5, 1.0], [1.0, 0.0]]  # 缓慢吸入再呼出 (同 PAD)
ENV_PLUCK = [[0.0, 0.0], [0.02, 1.0], [0.1, 0.0]] # 极其短促的撞击点 (同 PLUCK)

# ========================================================
# 3. 编写多轨矩阵编曲
# ========================================================
# 格式: [音符, 八度, 节拍, [角度, 距离], 包络线, 音色参数]

# 轨道 1: 虚空低频深渊 (Track Drone)
track_drone = [
    # 极低音 (八度-2)，长达 8 拍的和弦滑音！
    # 空间轨迹: 从正前方的极远处 (0, 15) 缓慢滑行包裹至脑后贴脸 (180, 0)
    [[(0, 4), (2, 5), (4, 7)], -2, 8.0, [(0, 180), (15, 0)], ENV_SWELL, CUSTOM_SUB_BASS],
    
    # 后 8 拍: 空间原路返回，和弦音高回落
    [[(4, 0), (5, 2), (7, 4)], -2, 8.0, [(180, 0), (0, 15)], ENV_SWELL, CUSTOM_SUB_BASS],
]

# 轨道 2: 空间跳跃的神经信号 (Track Glitch)
track_glitch = []
for i in range(16): # 循环生成 16 拍的极速琶音
    # 制造随机的左右耳跳跃感：偶数音符在极左(270)，奇数音符在极右(90)
    angle = 270 if i % 2 == 0 else 90
    
    track_glitch.extend([
        # 连续两个 16 分音符短促打击
        [0, 1, 0.25, [angle, 1.0], ENV_PLUCK, CUSTOM_GLITCH],
        [4, 1, 0.25, [angle, 1.0], ENV_PLUCK, CUSTOM_GLITCH],
        # 空白半拍产生顿挫律动
        [-1, 0, 0.5]
    ])

# 轨道 3: 贯穿全曲的太阳风暴扫频 (Track FX)
track_fx = [
    # 纯噪声不受音高影响(写0即可)。让风声从右后方(135)刮向左前方(315)
    [0, 0, 16.0, [(135, 315), (5, 1)], ENV_SWELL, CUSTOM_WIND]
]

# ========================================================
# 4. 混音与后处理
# ========================================================
print("🎛️ 乐谱解析完毕，开始并行多轨物理干涉运算...")
master_dry = synth.mix([track_drone, track_glitch, track_fx])

print("🌌 正在利用 FFT 卷积技术挂载史诗级 3D 大厅混响...")
# 房间尺寸拉到极大 (4.0)，吸收掉较多高频 (0.7) 营造深空恐惧感
final_master = synth.apply_reverb(master_dry, room_size=4.0, damping=0.7, wet_level=0.45)

print("🎧 请戴上耳机！正在播放最终母带...")
synth.play(final_master)
print("✨ 运算结束。")

# 可选：如果已安装 soundfile，可随时取消注释导出无损文件
# import soundfile as sf
# sf.write("Singularity_Master.flac", final_master, synth.sample_rate)
```
